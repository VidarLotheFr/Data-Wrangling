# Data-Wrangling

Personal Data Wrangling repository for storing Jupyter Notebooks (`.ipynb` files) used for exploratory analysis, data cleaning, and training purposes.

## Folder Structure

| Folder | Description |
|--------|-------------|
| `exploratory/` | Exploratory Data Analysis (EDA) notebooks |
| `cleaning/` | Data cleaning and preprocessing notebooks |
| `analysis/` | In-depth analysis and findings notebooks |
| `datasets/` | Raw and processed datasets used in the notebooks |

## Purpose

This repository serves as:
- Personal storage for data wrangling notebooks
- Reference material for training and learning
- A portfolio of exploratory analysis work
